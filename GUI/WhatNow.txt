This is a basic tutorial designed to help you get used to the UI in rak.


-	The Top Bar	-

All of your most important things are in the top bar. 
Here you can find both these tutorials, documentation for the engine, new releases for the engine as well as the source code. 

Some things that the top bar does and how to do them: 

Importing:
	File -> Import

Opening files: 
	File -> Open

Saving files: 
	File -> Save

Adding camera:
	Add -> Camera

Adding light 
	Add -> Point/Spot Light
